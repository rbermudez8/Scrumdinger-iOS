# Setup
1. Clone the repository:
-       git clone https://github.com/your-username/scrumdinger.git
  
2. Open the project in Xcode:
-       cd scrumdinger
-       open Scrumdinger.xcodeproj

3. Build and run the app on your simulator or device.
